# CallbackType

Callback type: 'bash_script' for shell commands, 'sub_agent' for spawning sub-agents, 'send_message' for injecting messages


## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `CallbackTypeBashScript`  | bash_script               |
| `CallbackTypeSubAgent`    | sub_agent                 |
| `CallbackTypeSendMessage` | send_message              |
