# SessionType

Session type:
- 'main': Root-level user interactions
- 'forked': User-created conversation branches
- 'subagent': Delegated task workers


## Values

| Name                  | Value                 |
| --------------------- | --------------------- |
| `SessionTypeMain`     | main                  |
| `SessionTypeForked`   | forked                |
| `SessionTypeSubagent` | subagent              |
